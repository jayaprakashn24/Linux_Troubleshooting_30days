Day 1: Which command checks disk usage?
Answer: df
💡 Explanation:
df (Disk Free) shows how much disk space is used and available on your system.
It displays information about file systems, including total size, used space, and free space.
🔍 What the other options do:
ls → Lists files and directories in a folder
ps → Shows currently running processes
top → Displays real-time system processes and resource usage
